# Cowspeak

Cow used to say, now it speak.

## Installation

Add this line to your application's Gemfile:

    gem 'cowspeak'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install cowspeak

## Usage

```
cowspeak let me sing you a song
```
